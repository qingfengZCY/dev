# 程序员的笔记软件，应该满足下面几个条件。


（1）跨平台，同时支持桌面电脑（Windows，Mac，Linux）和手机（Android，iOS）。

（2）随时同步，打开任何一台机器，都能接着上一次的工作继续写。

（3）实时存储，如果软件突然关闭，也不会丢失内容。

（4）支持 Markdown 格式，便于后期直接发布。

（5）支持推送到远程 Git 仓库，产生历史版本，同时作为远程备份。
